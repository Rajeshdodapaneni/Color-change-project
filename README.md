# Color-change-project
Color Change Web App: Built an interactive JavaScript application where users input a color name/HEX code, and the selected color dynamically updates a display block using DOM manipulation.
This project is a simple and interactive JavaScript-based Color Change Application where users can input any valid color name or HEX code, and the selected color is instantly applied to a display block. It demonstrates basic DOM manipulation, event handling, and real-time UI updates using vanilla JavaScript.
